Превращайте картинки PNG/JPG/BMP в слои группы винилов Forza Horizon 6. Программа выполняет генерацию, предпросмотр и импорт в одном окне. Обычному пользователю не нужны Python, папка `.venv`, bat-файлы или ручной ввод адресов памяти.

> **Как скачать EXE:** скачайте `forza-painter-fh6-v1.9.5.exe` со страницы [Releases](https://github.com/bvzrays/forza-painter-fh6/releases) и просто запустите его.

> **Маркет пресетов:** общие картинки, пресеты и JSON-пакеты можно смотреть на https://painter6.com или через новый баннер маркета внутри программы.

> **Если результат мутный:** сначала увеличьте «Случайные выборки». Значения больше **200000** обычно заметно повышают качество; ещё больше — ещё чище, но генерация длится гораздо дольше.

> **Импорт может занять время:** начиная с v1.4.1 программа перебирает несколько алгоритмов поиска шаблона FH6 и может потратить до 5 минут, чтобы безопасно найти таблицу слоёв. Всё это время держите FH6 в редакторе группы винилов, не переключайте меню. Если всё равно не выходит — экспортируйте подробный журнал.

| Что делает | Подробности |
| --- | --- |
| Генерация JSON | Превращает картинки в геометрический JSON во встроенном генераторе на GPU/OpenCL. |
| Предпросмотр | Показывает исходную и сгенерированную геометрию прямо в окне программы. |
| Импорт в FH6 | Импортирует JSON в открытый редактор групп винилов FH6. |
| Безопасная работа с FH6 | Автоматически находит и проверяет редактируемую таблицу слоёв перед записью. |
| Импорт/экспорт полных фигур | Экспериментальный импорт type-code на вкладке «Импорт» и экспорт текущей группы на вкладке «Экспорт». |
| Маркет пресетов | Открывает https://painter6.com из программы, чтобы просматривать общие картинки, пресеты и JSON-пакеты. |
| Проверка обновлений | Проверяет новые версии при запуске и показывает заметки из списка изменений. |

## Быстрый старт

1. Скачайте `forza-painter-fh6-v1.9.5.exe` со страницы [Releases](https://github.com/bvzrays/forza-painter-fh6/releases).
2. Положите EXE в обычную папку с правом записи, например `Рабочий стол\forza-painter-fh6`.
3. Запустите EXE двойным кликом. Для импорта в FH6 запускайте от имени администратора, если Windows не даёт доступ к процессу игры.
4. В FH6 откройте создание группы винилов / редактор группы винилов, загрузите шаблон из сфер, затем разгруппируйте его.
5. В программе сгенерируйте JSON, откройте вкладку «Импорт», введите точное количество слоёв шаблона и импортируйте.

Не скачивайте автоматический ZIP-архив «Source code» с GitHub, если только вы не разработчик проекта. Обычному пользователю нужен только `.exe-файл`.

## Генерация JSON

1. Откройте вкладку «Генерация JSON».
2. Нажмите «Добавить изображения» и выберите PNG/JPG/BMP.
3. Выберите пресет качества.
4. По желанию: включите «Использовать пользовательские настройки», чтобы поменять число выходных слоёв, разрешение, случайные и мутированные выборки.
5. Нажмите фиксированную нижнюю кнопку «Генерировать с текущими настройками».
6. Дождитесь, пока обновятся предпросмотр и журнал.

Сгенерированные файлы сохраняются рядом с исходной картинкой, например `image.500.json`, `image.1000.json` и `image.3000.json`.

Из одной картинки может получиться несколько файлов JSON — контрольных точек. Берите JSON с наибольшим числом слоёв под ваш шаблон; например, с шаблоном на 3000 слоёв используйте `image.3000.json` или финальный `image.json`. Импорт 500-слойного JSON в 3000-слойный шаблон будет мутным.

| Пресет качества | Выходные слои | Случайные выборки | Назначение |
| --- | ---: | ---: | --- |
| extremely fast | 500 | 30000 | Быстрая прикидка композиции |
| fast | 1000 | 60000 | Быстрые рабочие черновики |
| balanced | 1800 | 120000 | Рекомендуется по умолчанию |
| slow | 2500 | 220000 | Финальное качество; выходит в диапазон 200k+ |
| super slow | 3000 | 350000 | Наилучшая чёткость, очень медленно |

## Импорт JSON

1. Запустите FH6 и держите редактор группы винилов открытым.
2. Загрузите или создайте шаблон из множества простых сферических слоёв.
3. Разгруппируйте шаблон и запомните точное число слоёв в игре.
4. В программе откройте вкладку «Импорт», нажмите «Обновить» и выберите процесс `forzahorizon6.exe`.
5. Введите точное число слоёв шаблона.
6. Добавьте сгенерированный `.json` или нажмите «Использовать сгенерированный JSON».
7. Оставьте расширенные поля адресов пустыми и нажмите «Импорт JSON».

Для корректного сохранения обложки и применения границ FH требует 4 дополнительных граничных слоя. Пример: 1000-слойный JSON требует шаблон минимум на 1004 слоя; 3000-слойный шаблон позволяет импортировать около 2996 отображаемых фигур.

## Экспериментальные импорт/экспорт полных фигур

В v1.8.1 добавлена новая вкладка «Региональная покраска» для постепенной прорисовки по областям; в v1.8.0 добавлена поддержка исследовательских type-code JSON FH6 прямо в основном потоке работы: импорт полных фигур/type-code JSON — на вкладке «Импорт», экспорт текущей редактируемой группы FH6 — на вкладке «Экспорт».

Это нужно только для JSON полных фигур. Обычный сгенерированный JSON с эллипсами по-прежнему импортируйте через обычную вкладку «Импорт».

- Экспорт только читает данные и перед записью JSON проверяет редактируемую таблицу слоёв FH6.
- Импорт сам распознаёт JSON полных фигур/type-code в общем списке и ожидает сохранённый/повторно открытый шаблон из простых белых кругов, обычно 3000 слоёв, разгруппированный перед импортом.
- Поддерживаемые маркеры JSON полных фигур: `type_word`, `shape_word`, полные коды типов FH6, поля рукотворного JSON Kloudy/Fabric, поля font-shape и простые имена примитивов вроде `Circle`, `Square`, `Triangle`, `Ellipse`.
- Предпросмотр type-code JSON использует встроенные ресурсы вершин винилов FH6, поэтому многофигурные JSON из нового экспортёра и рукотворные JSON в стиле Kloudy/Fabric можно смотреть без откатов к прямоугольникам.
- Слои-маски предпросматриваются как прозрачные вырезы поверх уже нарисованных нижних слоёв; миниатюры интерфейса FH6 всё равно могут показывать саму фигуру маски.
- После импорта полных фигур сохраните и переоткройте группы винилов перед оценкой результата: FH6 может показывать устаревшие ресурсы живого шаблона в текущем редакторе.
- Если импорт/экспорт полных фигур не удаётся, программа прямо напоминает о важных проверках FH6/редактора/шаблона. «Экспорт отчёта полных фигур» используйте, только когда нужно отправить отладочные данные.
- Импортёр записывает только стабильные визуальные поля и 16-битное слово фигуры по смещению слоя `0x7A`.
- Он не копирует нестабильные указатели ресурсов FH6, например `0xA8`.
- После завершения импорта сохраните и перезагрузите группу винилов FH6 перед оценкой устаревших миниатюр интерфейса.

## Экспериментальная «Региональная покраска»

«Региональная покраска» — это режим постепенной прорисовки: сначала генерируется базовый проход по всему изображению, а затем можно выделять области (инструментами «Прямоугольник» или «Эллипс») и уточнять только их дополнительными слоями.

- Добавьте одно изображение, выберите профиль качества (он задаёт общий бюджет из `stopAt`) и настройте слои первого прохода и слои региона.
- Нажмите «Начать первый проход», чтобы сгенерировать базовые слои. Предпросмотр появится на правом холсте.
- Используйте инструмент «Прямоугольник» или «Эллипс» на левом холсте, чтобы выделить область. Красное наложение показывает выделение.
- Поставьте галочку «Режим исключения», чтобы рисовать зоны исключения (полупрозрачное **чёрное** наложение). Кнопкой «Переключить включение/исключение» меняйте режим выбранной фигуры. Если зоны включения и исключения пересекаются, исключение побеждает: «Покрасить выделенную область» сгенерирует слои везде, *кроме* исключённых зон.
- Нажмите «Покрасить выделенную область», чтобы добавить больше слоёв только внутри этой области. Повторите для каждой области.
- После всех проходов используйте «Открыть папку результатов» или «Сохранить результат JSON», чтобы получить финальный `base.json`.
- Импортируйте итоговый JSON через вкладку «Импорт» — тот же поток работы, что и при обычной генерации.
- Оставшийся бюджет показан рядом с «Остаток». Каждый региональный проход расходует слои из этого бюджета.

## Важные правила

- Шаблон FH6 должен быть разгруппирован перед импортом.
- Число слоёв в программе должно в точности совпадать с игрой.
- Не переключайте меню игры во время импорта.
- После перезапуска FH6, перезагрузки шаблона или изменения числа слоёв импортируйте заново с новым корректным числом.
- Если в JSON слоёв меньше, чем в шаблоне, неиспользуемые слои шаблона скрываются.
- Если в JSON слоёв больше, чем в шаблоне, лишние фигуры обрезаются.
- Прозрачный фон PNG не импортируется как видимый фон.

## Файлы среды выполнения

EXE одним файлом временно извлекает внутренние файлы и хранит обычные данные среды выполнения вне EXE. Программа показывает точные пути в журнале запуска.

Ожидаемые внешние папки рядом с EXE:

- `runtime/`: журналы, сгенерированные данные сессии и временные файлы программы.
- `webui-data/`: локальный кэш браузера/интерфейса.

Эти папки можно удалить при закрытой программе, если хотите сбросить локальные данные среды выполнения.

## Устранение неполадок

- **EXE не импортирует в FH6:** закройте программу и запустите EXE от имени администратора.
- **Ошибка GPU/OpenCL:** обновите драйверы NVIDIA/AMD/Intel. Встроенный генератор использует OpenCL.
- **Шаблон не находится:** убедитесь, что вы в редакторе групп винилов, шаблон разгруппирован, число слоёв точное, а меню не менялось во время сканирования.
- **Результат мутный:** используйте JSON с бóльшим числом слоёв или поднимите «Выходные слои» / «Случайные выборки».
- **Нужна помощь с отладкой:** используйте «Экспорт подробного журнала» в программе и приложите журнал к issue.
